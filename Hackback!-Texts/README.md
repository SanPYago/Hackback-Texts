# Hackback!-Texts

A curated reader of HackBack!-related texts by Phineas Fisher / Phineas Phisher.

This repository is intended to be read like a short book: a chronological compilation of public texts related to HackBack!, hacktivism, surveillance, and political hacking.

## Reading order

1. [Hackback - A DIY GUIDE II](texts/01-hackback-a-diy-guide-ii.md)
2. [Hackback - A DIY GUIDE 1](texts/02-hackback-a-diy-guide-1.md)
3. [Hack Back](texts/03-hack-back.md)

## Purpose

The purpose of this repository is to preserve and organize these texts as a readable compilation.

This is not an operational hacking manual. The texts are included or linked for historical, political, and cybersecurity research.
